# Console Snake Game

Paste the contents of `snake.js` into your browser DevTools console on any page.

## Controls
- **Arrow keys**: Move
- **Space**: Start / restart after game over
- **P**: Pause / resume
- **Ctrl+E**: Hide/show game window

## Features
- Classic snake movement and collision behavior
- Score + persistent best score (`localStorage`)
- Configurable apple count and speed
- Resizable, draggable game window
- Game over and pause overlays
